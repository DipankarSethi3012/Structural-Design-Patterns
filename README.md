# Structural Design Patterns

## Introduction
Design patterns are proven solutions to common problems in software design. Among different categories of design patterns, structural design patterns focus on how classes and objects can be composed to form larger structures. These patterns help you ensure that if one part of a system changes, the entire system doesn't need to do the same.
